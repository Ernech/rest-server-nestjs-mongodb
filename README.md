# Nest Js Rest server with MoongoDB
1. Copy the ```.env.example``` and rename it to ```.env```
2. Set your enviroment variables in the ```.env``` file (Set the ```dev``` value to the NODE_ENV env variable).
3. Execute 
```
npm install 
```
4. Execute project.
```
npm run start:dev
```
